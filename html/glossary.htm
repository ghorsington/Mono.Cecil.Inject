I

Injection, Method injection

The process of inserting a call to a static method into another method.

Injector

An instance of InjectionDefinition that performs the injection.

Injector creator

A method that creates an injector.

Hook, Hook method, Injector method

A static method that will be called by the injection target.

Hook target, Injector target, Target

A method (or a class) that is being injected into.

M

Method redirection

A special injection in which the injection method can replace the target method's return value or prevent the target's code from executing.
